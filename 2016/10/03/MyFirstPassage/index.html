#静态博客搭建

#一、操作过程

##准备工作

  1. git for windows
  2. node.js
  3. 注册github账号

##对git的部署

####首先将git添加到环境变量中

####将注册的邮箱和用户名上传

$git config--global user.name "Your name"

$git config--global user.name "email@example.com"

####利用如下语句生成秘钥和公钥，然后打开公钥文件将其中的内容全部复制到github账号的ssh key中
$ssh-keygen -t rsa -C youremail@example.com

####检查是否可以连接
ssh git@github.com

####确定连接成功后执行克隆远程仓库和提交本地仓库的操作

##HEXO

####首先安装hexo
npm install hexo -g

####新建一个目录blog
mkdir blog

####转移到目录blog下，并对hexo进行初始化
cd bolg
hexo init

####启动hexo server
hexo server

在浏览器中访问localhost:4000
如果能够成功访问则说明hexo配制成功

##将本地hexo与远程github关联

####修改配置文件
在blog中找到配置文件_config.yml,用记事本或其他编辑器打开并对其进行如下修改

deploy:
type: git
repository: git@github.com:yruri/yruri.github.io.git
branch: master

####安装hexo-deployer-git

npm install hexo-deployer-git -S

####生成静态网页

hexo g

####部署到远程github

hexo d

(hexo g和hexo d是hexo generate 与hexo deploy的缩写)

##设置主题

####安装
$ git clone https://github.com/huangjunhui/concise.git themes/concise

####启用
修改你的博客根目录下的config.yml配置文件中的theme属性，将其修改为concise。

####更新
$ cd themes/concise
$ git pull
更新前，最好先将themes/concise/_config.yml文件备份
